## 在“价格发现”阶段抢先布局的机会

## 常见问题 FAQ

**Q1：这些项目的代币经济是否真的会大幅稀释早期份额？**  
A：大多数采用 **线性释放（Cliff → Linear）** 模式，核心团队和 VC 通常在 6-12 个月后才开始解锁，早期流通盘低于 8%-15%，因此初期波动极大，但也意味着暴涨前没有巨量抛压。

**Q2：我英文一般，该如何持续追踪官方任务？**  
A：可以把 Discord 加入“静音但允许推送”，每天固定 10 分钟扫置顶频道和 **公告栏**；关键词如“Zealy”或“Galxe”出现，基本是空投任务，截图+钱包地址即可一键完成。

**Q3：DePIN 项目节点是否必须购买硬件？**  
A：现阶段 90% 节点支持云服务器，只要配置 4G 内存 + 100G SSD 就能跑。唯一需要考虑的是电费与 IP 稳定性，香港/新加坡机房性价比较高。

**Q4：如何辨别“智商税”式骗局？**  
A：先看团队 GitHub 提交频率；再搜主网 GitHub 是否开源，能否查看合约审计报告；最后看 X/Twitter KOL 推特互动质量，沉默此处最容易暴露跑路红标。

**Q5：怎样才能快人一步拿到白名单名额？**  
A：两类方法并行：1）官方 Discord“哭穷”频道每天蹲守 GM、devchat 内测群；2）使用链上 **蓝筹 NFT 做抵押** 可直享 LBP 双倍倍数，提前确认所持 NFT 是否列入官方白名单。

**Q6：SOL 生态与 BTC 生态的升值空间哪个更大？**  
A：SOL 侧重开发者/用户体量，流量空间大；BTC 生态靠叙事推动，门槛高但溢价也高。建议相对均衡：DePIN 类项目别排斥任何链，哪条链空投多就放主精力在哪个链。
